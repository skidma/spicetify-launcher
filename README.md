## The Problem

Have you ever experienced Spicetify mysteriously disappearing every time you restart Spotify? This launcher solves that exact issue by ensuring Spicetify stays applied consistently.

## Features

- **Automatic Re-application**: Keeps Spicetify active even after Spotify updates
- **Lightweight**: Minimal system resource usage
- **Safe**: Doesn't modify core Spotify files beyond standard Spicetify application


### Installation
1. Download the latest release depending on the version you choose.
2. Extract the files to your preferred location.
3. Run the setup script/executable and enjoy
